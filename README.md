# 💈 Barbearia do Corte

Landing page moderna e responsiva para uma barbearia, desenvolvida com **HTML, CSS e JavaScript puro** (sem frameworks ou bibliotecas externas). O site foi pensado para converter visitantes em clientes através de um agendamento rápido pelo WhatsApp.

## 🔗 Demo

> Abra o arquivo `index.html` no navegador para ver o projeto rodando localmente (veja a seção [Como executar](#-como-executar)).

## ✨ Funcionalidades

- **Hero animado** com imagem de destaque e chamada para ação
- **Contadores animados** (clientes atendidos, anos de experiência, satisfação)
- **Seção Sobre** apresentando a barbearia
- **Catálogo de serviços** com preços e botão de agendamento direto no WhatsApp para cada um
- **Diferenciais** do negócio em formato de grid
- **Galeria de fotos** com efeito *lightbox* (clique para ampliar)
- **Carrossel de depoimentos** de clientes
- **Seção "Como funciona"** explicando o processo de agendamento em 4 passos
- **Mapa incorporado** (Google Maps) e informações de contato
- **Botão flutuante do WhatsApp** e botão "voltar ao topo"
- **Menu responsivo** (hamburguer) para dispositivos móveis
- Cursor customizado e animações de *scroll reveal*

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização e responsividade (arquivo `style.css`)
- **JavaScript** — interatividade: menu, contadores, lightbox, carrossel, etc. (arquivo `script.js`)
- **Google Fonts** — tipografia (Playfair Display, Inter, DM Mono)

## 📁 Estrutura do projeto

```
barbearia_do_corte/
├── index.html    # Estrutura e conteúdo da página
├── style.css     # Estilos e responsividade
└── script.js     # Interatividade (menu, contadores, galeria, etc.)
```

## 🚀 Como executar

Como é um projeto **100% front-end**, não é necessário instalar nada. Basta:

1. Clonar o repositório:
   ```bash
   git clone https://github.com/RondneyLoiola/barbearia_do_corte.git
   ```
2. Entrar na pasta do projeto:
   ```bash
   cd barbearia_do_corte
   ```
3. Abrir o arquivo `index.html` diretamente no navegador.

> 💡 Dica: se estiver usando o VS Code, a extensão **Live Server** facilita bastante — basta clicar com o botão direito em `index.html` e escolher "Open with Live Server".
